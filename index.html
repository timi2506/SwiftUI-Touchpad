Usage

TouchpadView

TouchpadView(buttons: [
    TouchpadButton(id: "button1", text: "Button1", symbolName: "apple.logo", boundaries: CGRect(x: 100, y: 100, width: 100, height: 100), action: {
        print("button 1 pressed")
    })
], touchPadLabel: "Touchpad")

Recommended Usecases:

  • ✅ in NavigationLinks
  • ✅ in Sheets
  • ❌ inside another View

TouchpadButton

TouchpadButton(id: "button1", text: "Button1", symbolName: "apple.logo", boundaries: CGRect(x: 100, y: 100, width: 100, height: 100), action: {
    print("button 1 pressed")
})

Properties

  • id (Required) - Unique identifier for each button.
  • text (Semi-Optional) - Button's text label.
  • symbolName (Semi-Optional) - SF Symbol name for the button image.
  • boundaries (Required) - CGRect defining button placement and tap area.
  • action (Semi-Optional) - Action executed when tapped.
  • border (Required) - Toggles visibility of the tap area border.
  • borderColor (Optional, Default: red) - Border color.
  • borderRadius (Optional, Default: 0) - Corner radius of the border.
